Difference between var,let and const

var:

Var is global or functional scoped

Var can be redeclared

Var can be reassigned

Var can be modified later

Let:

Let is a block scoped

Let cannot be reassigned

Let cannot be redeclared

Let cannot be modified later

Const:

Const is a block scoped

Let cannot be reassigned

Let cannot be redeclared

Let cannot be modified later

Hoisting is a default Javascript mechanism where our variables and function declarations are moved to the top of their scope before code execution()
